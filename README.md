## DiffWind是什么
DiffWind是一个量化投资系统，主要包括基于指标的筛选模型（案例学习部分因为不成熟并没有开源）。

## DiffWind如何使用
DiffWind使用的数据源：  
**网易：**  
财务报表历史数据
日线交易数据

**东方财富：**  
资金流、沪深股通资金流数据
融资融券历史数据
大宗交易数据
其他特色数据

**新浪：**  
行业分类数据
分红配股数据（用于复权计算）

**同花顺：**  
分红、财务数据等

**雪球（因发爬虫问题已弃用）：**  
财务报表历史数据
股本结构历史数据


**使用步骤：**
1. 安装数据库：PostgreSQL 9.x以上 
2. 执行src/main/sqlscripts下的SQL脚本文件，创建表和索引  
3. DiffWind初次执行时需准备历史数据，初始化数据库(只需执行一次，以后不需要再执行)：  
   DiffWindFirstTime  
4. 执行筛选模型：  
   DiffWindMain
